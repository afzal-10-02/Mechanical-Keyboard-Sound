# 🎹 Mechanical Keyboard Sound Extension

**Enhance your VS Code typing experience with realistic mechanical keyboard sounds!** 🔊

---

## ✨ Features
✅ Plays mechanical keyboard sounds when you type in VS Code.    
✅ Supports all keys, including **space, enter, backspace**, and more.  
✅ Minimal performance impact with real-time sound playback.  

---

## 🚀 Installation

### **From VS Code Marketplace** (Recommended)
1. Open **VS Code**.
2. Go to **Extensions** (`Ctrl+Shift+X`).
3. Search for **Mechanical Keyboard Sound**.
4. Click **Install**.

## 🛠️ How It Works
This extension listens to **keystrokes** in VS Code and plays corresponding **mechanical keyboard sound effects** using the `node-web-audio-api` module.
